# MEMORY.md — Working Memory Template

This is your active long-term memory. It gets consolidated when it grows too large.

## Template

```markdown
# Working Memory

## Recent Decisions
- [Date] Decision made and why

## Active Projects
- Project name: status, next steps

## Learned Preferences
- User prefers: [preference]

## Context Worth Remembering
- Important context that spans sessions
```

## Consolidation Rules

When this file exceeds 4KB:
1. Old content moves to `archive/YYYY-MM-DD.md`
2. Active/recent content stays
3. Key decisions and patterns are preserved
4. ESSENTIAL.md is never touched
